<h2 style='width:100%;text-align:center'>Features</h2>

- [x] Responsive Design
- [x] Add blocks of Text (headings, simple text...) using commands
- [x] Edit blocks
- [x] Delete blocks
- [x] Filter commands
- [ ] Add block based on the location of where you called the command
- [ ] More block types
  - [x] Checkbox
  - [x] Divider
